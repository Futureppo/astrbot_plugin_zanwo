# astrbot_plugin_zanwo

[![GitHub](https://img.shields.io/badge/作者-Futureppo-blue)](https://github.com/Futureppo)


## 插件简介
**名片点赞插件**，用户发送指令 `赞我` 后，自动向用户发送最多 50 次名片赞。


## 使用方法

### 触发指令
在聊天窗口发送以下指令即可触发点赞：
```text
赞我
```

## 注意事项

1. **点赞限制**  
   - 非好友每天只能点50人，每人50个。

2. **可能错误**  
   - 用户隐私设置未开启陌生人点赞权限时，直接提示失败。
