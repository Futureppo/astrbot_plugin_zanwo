## 📦 插件简介

遵循Onebotv11协议，给QQ用户点赞，每天可一次性点50个赞

## ⌨️ 使用方法

在聊天窗口发送以下指令即可触发点赞：

```text
赞我
```

## 📌 注意事项

- 点赞限制：非好友每天只能点50人，每人50个。
- 可能错误：用户隐私设置未开启陌生人点赞权限时，直接提示失败。
